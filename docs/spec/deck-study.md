# デッキ学習 (Deck Study)

選択中デッキ内で、セッションキューが **覚えた** 操作で減っていく **セッション学習** トラック。

- **DB 更新なし** — `review_state` / スマート学習に影響しない
- **study_events 記録あり** — セッション・覚えた/まだは [study-metrics.md](./study-metrics.md) に従い永続化
- タブ: **デッキ学習**（`AppTab.deckStudy`）
- 詳細索引: [study.md](./study.md)

## 画面構成

### デッキ学習ハブ

- ホームで選択中デッキの **Coverflow**（中央クリック → カードプレビュー）
- **4 つの手段**（フラッシュカード / 書く / テスト / マッチ）— **`learn` は含まない**
- カード追加バー、カード一覧（**ドラッグ並べ替え**可）、シャッフル toggle
- カード一覧・Coverflow の表示順は `sort_order`（[library.md](./library.md) / [data-model.md](./data-model.md)）
- トップバー: 選択デッキ名、カード検索
- 縦スクロール可（スクロールバー非表示）

### セッション（共通）

- 手段起動で全画面セッション
- 残り 0 で **学習完了** 画面
- カードプレビュー（1 枚）はセッションキュー外・SRS 非連動

### フラッシュカード（Quizlet 風ラウンド）

ラウンド単位で **覚えた** / **もう一度（次の周）** に選別する。

| 項目 | 仕様 |
|------|------|
| 進捗バー | **この周でスワイプ済み枚数 / この周の枚数**。1 枚目表示中は **0 / N**（バー 0%） |
| 選別 | 答え表示後: **覚えた**（右スワイプ / `2`）/ **もう一度**（左スワイプ / `1`） |
| 周回 | 「もう一度」は **即末尾へ回さず**、その周のリストに蓄積。周が終わったら結果画面 |
| 結果画面 | 今回覚えた枚数・残り枚数・累計 + **円グラフ**（shadcn Chart / recharts）。残り > 0 なら **次の周へ** |
| 完了 | 残り 0 でセッション完了。「もう一度」で新規セッション |
| 下書き | `localStorage`（`xanki.flashcard-session.v1:{deckId}`）。途中で戻っても同デッキ再入で再開。削除カードは突合して除外。完了 / 明示リセットで削除 |
| metrics | 途中 exit で metrics セッションは完了。再開時は新規 `study_sessions`（キュー下書きとは独立） |

キーボード: `1` = もう一度, `2` = 覚えた（スマート学習タブの 1–4 と独立）。Space / クリックで答え toggle。

### 他手段（書く / テスト / マッチ）

- 進捗: **残り N / M**（従来どおり）
- 書く / テストはマスクありカードのみ等、各 Mode の仕様に従う

## 手段一覧

| 手段 | コード | v1 キュー |
|--------|--------|-----------|
| フラッシュカード | `flashcards` | デッキ全件（ラウンド選別） |
| 書く | `write` | デッキ全件（マスクありのみ出題） |
| テスト | `test` | 同上 |
| マッチ | `match` | 同上 |

v1 範囲選択は **デッキ全件のみ**。

**並び:** シャッフル OFF 時は `listCards` の `sort_order` 順。シャッフル ON はセッション開始時の乱順（永続順は変えない）。

## 表示・操作（共通）

- フリップ / peek / 3D フリップ: [study.md](./study.md) §表示 および [leitner-study.md](./leitner-study.md) と同一コンポーネント
- Space / クリック: 答え toggle
- **AI に聞く**（フラッシュカード）: `.review-actions` に ghost ボタン → オーバーレイで質問入力 → SSE ストリーム表示（[cloud.md](./cloud.md) §9）
