# デッキとカード一覧

> **旧称**: 「ライブラリ」— 会話・コード内部名（`collectionRevision` 等）。**UI タブ名では使わない**。用語正本: [glossary.md](./glossary.md)

## 画面分担

| 画面 | 役割 |
|------|------|
| **ホーム** | デッキ一覧とデッキ操作のみ |
| **デッキ学習** | 選択中デッキのカード一覧（グリッド表示） |
| **Leitner学習** | 横断 due・ヒーローカード・デッキ別 due |

ホームで選んだデッキが、**デッキ学習**タブのカード一覧に反映される。

## デッキ選択（ホーム）

- **常に 1 つのデッキを選択**する
- 全デッキ横断表示（旧「すべて」）は **ない**
- 起動時: 最終使用デッキがあればそれを選択、なければ先頭デッキ
- 選択中デッキを削除した場合: 残りの先頭デッキを自動選択（デッキが 0 件なら未選択）
- **デッキ 0 件**（新規ユーザー等）: `GET /api/decks` 時にサーバーが **「デフォルト」** デッキを 1 件自動作成する（Web / デスクトップ共通）

## デッキ操作（ホーム）

| 操作 | 説明 |
|------|------|
| 作成 | デッキ一覧**の上**のフォーム — 名前入力（placeholder: 「新しいデッキの名前」）→ 追加 |
| リネーム | ✎ アイコン |
| 削除 | × — 確認ダイアログ後、**デッキと配下の全カードを同一 TX で論理削除**（`review_state` も削除） |
| エクスポート | ↓ — JSON ダウンロード |
| インポート | 作成フォーム横 — JSON ファイル |

## カード追加（デッキ学習タブ）

- カード一覧**の上**に **インライン作成エリア**（`TextMaskComposerEmbedded`）を表示（Web / Desktop 共通）
- マスクエディタ（`TextMaskEditor`）と同じ UI: テキスト入力 → 範囲選択 → **+ マスク** → 保存（⌘Enter 可）
- **Q** ボタンで一問一答形式に切替（問題文 + 解答）。**マスク** ボタンで Q モードを解除し、問題文と解答を改行で連結した本文に戻してマスク編集を続行
- 任意メモ欄 + **カードを保存** ボタン
- **テキスト取込 / スクショ取込**: サイドバーのグローバルショートカット（⌥⌘M / ⌥⌘S）のみ。ホームで選択中のデッキをエディタの初期値にする（`localStorage` + Rust `capture_deck_id`）
- 画像カード: スクショ取込フロー（OS 範囲選択）→ 別ウィンドウの画像マスクエディタ

## カード一覧（デッキ学習タブ）

- 見出し eyebrow: **カード**
- 対象: **ホームで選択中のデッキのみ**（検索も同一デッキ内）
- カード追加・削除時: タイル **enter/exit**（scale + fade）。削除後は残タイルが **layout position** で詰まる（Motion `LayoutGroup` + `popLayout`）
- タイル形状: 横長（aspect-ratio 4:3）。種別バッジなし。プレビュー中央揃え（テキストは省略・左揃え・縦中央、画像は contain + inset 影・マスクオーバーレイ表示）
- カード一覧エリアのスクロールバーは非表示（スクロール自体は可能）
- テキスト: マスク付きプレビュー
- Q&A: 問題文（マスク適用プレビュー）。解答は非表示（「解答あり」表示）
- 画像: 保存 PNG + マスクオーバーレイ（マスクごとの `color` を反映）
- 種別バッジ（Text / Image / **Q&A**）

## カード操作

| 操作 | 説明 |
|------|------|
| カード本体クリック | フラッシュカードプレビュー（学習ハブ）またはマスクエディタ再編集 |
| 削除 | カード下部の「削除」のみ（複製・リセットは **なし**） |

### 削除確認

- **`window.confirm` は使わない**（Tauri WebView で動作しない）
- アプリ内モーダル（`confirm-backdrop` / `confirm-dialog`）
- Esc でキャンセル

## 同期

- SSE `revision` 受信で一覧 refetch（`collection-sync` / `library-sync-controller`）
- マスクエディタ保存後 REST mutation → SSE

## 受け入れ条件

- [ ] ホームでデッキを選ぶと、学習タブにそのデッキのカードが表示される
- [ ] 削除確認 → 削除する、でカードが消える
- [ ] 新規保存後、デッキ一覧の件数とカードグリッドが更新される
- [ ] カードクリックでプレビューまたは再編集エディタが開く
