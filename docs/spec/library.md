# ライブラリ / カードコレクション

## 画面分担

| 画面 | 役割 |
|------|------|
| **Home** | デッキ一覧とデッキ操作のみ |
| **学習** | 選択中デッキのカードコレクション（グリッド表示） |

Home で選んだデッキが、学習タブのカード一覧に反映される。

## デッキ選択（Home）

- **常に 1 つのデッキを選択**する
- 全デッキ横断表示（旧「すべて」）は **ない**
- 起動時: 最終使用デッキがあればそれを選択、なければ先頭デッキ
- 選択中デッキを削除した場合: 残りの先頭デッキを自動選択（デッキが 0 件なら未選択）

## デッキ操作（Home）

| 操作 | 説明 |
|------|------|
| 作成 | 名前入力 → 追加 |
| リネーム | ✎ アイコン |
| 削除 | × — 確認ダイアログ後、デッキと中のカードを論理削除 |
| エクスポート | ↓ — JSON ダウンロード |
| インポート | サイドバー下部 — JSON ファイル |

## カード一覧（学習タブ）

- 対象: **Home で選択中のデッキのみ**（検索も同一デッキ内）
- カード一覧エリアのスクロールバーは非表示（スクロール自体は可能）
- テキスト: マスク付きプレビュー
- Q&A: 問題文（マスク適用プレビュー）。解答は非表示（「解答あり」表示）
- 画像: 保存 PNG + マスクオーバーレイ（マスクごとの `color` を反映）
- スター toggl（★）
- 種別バッジ（Text / Image / **Q&A**）、Box 番号表示

## カード操作

| 操作 | 説明 |
|------|------|
| カード本体クリック | マスクエディタで再編集 |
| 削除 | カード下部の「削除」のみ（複製・リセットは **なし**） |

### 削除確認

- **`window.confirm` は使わない**（Tauri WebView で動作しない）
- アプリ内モーダル（`confirm-backdrop` / `confirm-dialog`）
- Esc でキャンセル

## 同期

- `library-changed` イベントで一覧再読込
- main ウィンドウフォーカス時も再読込
- マスクエディタ保存時に Rust から `library-changed` emit

## 受け入れ条件

- [ ] Home でデッキを選ぶと、学習タブにそのデッキのカードが表示される
- [ ] 削除確認 → 削除する、でカードが消える
- [ ] 新規保存後、デッキ一覧の件数とカードグリッドが更新される
- [ ] カードクリックで再編集エディタが開く
