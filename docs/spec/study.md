# 学習 (Study)

## 画面構成

### 学習ハブ（学習タブ・通常時）

- ホームで選択中のデッキの **Coverflow カルーセル**（「学習を始める」の上）。ドラッグで移動、中央クリックでフラッシュカードプレビュー。出現アニメーションなし
- ホームで選択中のデッキの **カード一覧**
- **5 つの学習モード起動ボタン**（フラッシュカード / **復習** / 書く / テスト / マッチ）
- カード追加バー（カード一覧の上）
- シャッフル toggl（フラッシュカード・書く・テスト向け）
- カード検索（トップバー）
- **学習ハブ全体が縦スクロール**（モード起動エリアは固定しない）。スクロールバーは非表示
- 学習タブは **アンマウントせず非表示**にする（他タブ往復時の再読み込みチラつき防止）

### 学習セッション（モード実行中）

- 起動ボタン押下で **メイン領域全体** がそのモード UI に切り替わる
- ハブ ↔ セッション切替時は **即時切替**（ペインアニメーションなし）
- **サイドバーは自動で閉じる**（全画面表示）
- トップバー「戻る」で学習ハブに復帰
- セッション開始前にサイドバーが開いていた場合、ハブ復帰時にサイドバーを再度開く
- サイドバー ☰ トグルで、セッション中もオーバーレイ表示可能
- **カード本体**（フラッシュカード / **復習** / 書く / テスト / マッチ）は、進捗・ヒントと操作ボタンの間で **画面中央** に配置する
- カードプレビュー（Coverflow 中央タップ → フラッシュカード 1 枚）も同じ中央配置

## モード一覧

| モード | コード | キュー | SRS | 概要 |
|--------|--------|--------|-----|------|
| フラッシュカード | `flashcards` | 全カード (`all`) | なし | Space / クリック 答え、→ で次へ |
| **復習** | `learn` | due のみ (`due`) | **あり** | Space / クリック 答え、1/2 で自己評価 |
| 書く | `write` | 全カード | なし | 穴埋め入力 |
| テスト | `test` | 全カード | なし | 4 択 |
| マッチ | `match` | 全カード | なし | 問題と答えのペア選択 |

デッキ指定フィルタあり。フラッシュカード/書く/テストはシャッフル可。

## 表示

### Q&A カード（`kind=qa`）

- 非表示: 問題文のみ。マスクありなら Chartreuse 塗り
- 答え表示: 問題文は **全マスク解除** + 解答ブロックを下に表示
- 詳細: [qa-cards.md](./qa-cards.md)

### テキストカード

- 非表示: マスク部分を Chartreuse 塗り + 黒枠
- 答え表示: 点線枠
- インタラクティブ時: マスク個別クリックでピーク（点線）。同じマスクを再クリックで再マスク（トグル）
- 表に戻したとき（答え非表示）: 個別ピーク状態はリセットし、全マスクを Chartreuse 塗りに戻す
- 復習・フラッシュカード: カードフリップアニメーション（**3D rotateY は CSS**、`prefers-reduced-motion` 時は切替のみ）
- フラッシュカード **次カード** 遷移: 即時切替（ペインアニメーションなし）
- マスク **全体 reveal**（flip 後）: 点線化を stagger（Motion）。個別 peek は tap scale
- 進捗バー幅: Motion spring
- フリップのクリック: テキスト選択・ドラッグ操作では発火しない

### 画像カード

- `ImageWithMaskOverlays` でマスク矩形
- 非表示: 不透明 Chartreuse
- ピーク/全体表示: 点線
- インタラクティブ時: テキストカードと同様、個別クリックでピーク ↔ 再マスクをトグル。表に戻したときはピークをリセット

## 復習モード (SRS) キーボード

| キー | 動作 |
|------|------|
| Space | 答え表示 / 非表示トグル |
| 1 | できなかった |
| 2 | できた |

## Leitner スケジューラ

| 箱 | 次回まで |
|----|---------|
| 1 | 当日（0 日） |
| 2 | 1 日 |
| 3 | 3 日 |
| 4 | 7 日 |
| 5 | 21 日 |

- 新規カード: box=1, `due_at=now`
- 「できた」: box+1（最大 5）
- 「できなかった」: box=1

Tray「今日の復習: N件」= `due_at <= now` の件数。クリックで **学習タブ** を開く。

## 受け入れ条件

- [ ] 学習ハブからモードを起動でき、戻るでハブに復帰できる
- [ ] モード開始でサイドバーが閉じる
- [ ] 復習モードで Space → 1/2 後、Tray 件数が更新される
- [ ] 画像/テキストともマスク位置がカードプレビューと一致する
- [ ] フラッシュカードは SRS に影響しない
