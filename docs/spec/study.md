# 学習 (Study)

## 画面構成

### 学習ハブ（学習タブ・通常時）

- Home で選択中のデッキの **カードコレクション**
- **5 つの学習モード起動ボタン**（フラッシュカード / 学習 / 書く / テスト / マッチ）
- シャッフル toggl（フラッシュカード・書く・テスト向け）
- カード検索（トップバー）

### 学習セッション（モード実行中）

- 起動ボタン押下で **メイン領域全体** がそのモード UI に切り替わる
- **サイドバーは自動で閉じる**（全画面表示）
- トップバー「戻る」で学習ハブに復帰
- サイドバー ☰ トグルで、セッション中もオーバーレイ表示可能

## モード一覧

| モード | キュー | SRS | 概要 |
|--------|--------|-----|------|
| フラッシュカード | 全カード (`all`) | なし | Space / クリック 答え、→ で次へ |
| 学習 | due のみ (`due`) | **あり** | Space / クリック 答え、1/2 で自己評価 |
| 書く | 全カード | なし | 穴埋め入力 |
| テスト | 全カード | なし | 4 択 |
| マッチ | 全カード | なし | 問題と答えのペア選択 |

デッキ指定フィルタあり。フラッシュカード/書く/テストはシャッフル可。

## 表示

### Q&A カード（`kind=qa`）

- 非表示: 問題文のみ。マスクありなら Chartreuse 塗り
- 答え表示: 問題文は **全マスク解除** + 解答ブロックを下に表示
- 詳細: [qa-cards.md](./qa-cards.md)

### テキストカード

- 非表示: マスク部分を Chartreuse 塗り + 黒枠
- 答え表示: 点線枠
- インタラクティブ時: マスク個別クリックでピーク（点線）
- 学習・フラッシュカード: カードフリップアニメーション（`prefers-reduced-motion` 時は切替のみ）

### 画像カード

- `ImageWithMaskOverlays` でマスク矩形
- 非表示: 不透明 Chartreuse
- ピーク/全体表示: 点線

## 学習モード (SRS) キーボード

| キー | 動作 |
|------|------|
| Space | 答え表示 / 非表示トグル |
| 1 | できなかった |
| 2 | できた |

## Leitner スケジューラ

| 箱 | 次回まで |
|----|---------|
| 1 | 当日（0 日） |
| 2 | 1 日 |
| 3 | 3 日 |
| 4 | 7 日 |
| 5 | 21 日 |

- 新規カード: box=1, `due_at=now`
- 「できた」: box+1（最大 5）
- 「できなかった」: box=1

Tray「今日の復習: N件」= `due_at <= now` の件数。クリックで **学習タブ** を開く。

## 受け入れ条件

- [ ] 学習ハブからモードを起動でき、戻るでハブに復帰できる
- [ ] モード開始でサイドバーが閉じる
- [ ] 学習モードで Space → 1/2 後、Tray 件数が更新される
- [ ] 画像/テキストともマスク位置がカードプレビューと一致する
- [ ] フラッシュカードは SRS に影響しない
