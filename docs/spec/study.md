# 学習 (Study)

## モード一覧

| モード | キュー | SRS | 概要 |
|--------|--------|-----|------|
| フラッシュカード | 全カード (`all`) | なし | Space で答え、→ で次へ |
| 学習 | due のみ (`due`) | **あり** | Space → 1/2 で自己評価 |
| 書く | 全カード | なし | 穴埋め入力 |
| テスト | 全カード | なし | 4 択 |
| マッチ | 全カード | なし | 問題と答えのペア選択 |

デッキ指定フィルタあり。フラッシュカード/書く/テストはシャッフル可。

## 表示

### Q&A カード（`kind=qa`）

- 非表示: 問題文のみ。マスクありなら Chartreuse 塗り
- 答え表示: 問題文は **全マスク解除** + 解答ブロックを下に表示
- 詳細: [qa-cards.md](./qa-cards.md)

### テキストカード

- 非表示: マスク部分を Chartreuse 塗り + 黒枠
- 答え表示: 点線枠
- インタラクティブ時: マスク個別クリックでピーク（点線）

### 画像カード

- `ImageWithMaskOverlays` でマスク矩形
- 非表示: 不透明 Chartreuse
- ピーク/全体表示: 点線

## 学習モード (SRS) キーボード

| キー | 動作 |
|------|------|
| Space | 答え表示 / 非表示トグル |
| 1 | できなかった（答え表示後） |
| 2 | できた（答え表示後） |

## Leitner スケジューラ

| 箱 | 次回まで |
|----|---------|
| 1 | 当日（0 日） |
| 2 | 1 日 |
| 3 | 3 日 |
| 4 | 7 日 |
| 5 | 21 日 |

- 新規カード: box=1, `due_at=now`
- 「できた」: box+1（最大 5）
- 「できなかった」: box=1

Tray「今日の復習: N件」= `due_at <= now` の件数。

## 受け入れ条件

- [ ] 学習モードで Space → 1/2 後、Tray 件数が更新される
- [ ] 画像/テキストともマスク位置がライブラリプレビューと一致する
- [ ] フラッシュカードは SRS に影響しない
