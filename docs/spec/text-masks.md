# テキストマスク

## 概要

1 テキスト = 1 カード。マスクは複数可。分割保存はしない。

## エディタ UI

- 取得テキストを **textarea + 背景レイヤ** で表示
- ドラッグで範囲選択 → フローティング「+ マスク」（Q はツールバー。詳細は [qa-cards.md](./qa-cards.md)）
- マスク部分: **Chartreuse 背景 (`--accent-soft`) + 黒下線**（レイアウトに影響しない inset shadow。選択ズレ防止）
- 背景レイヤと textarea は **同一スクロールコンテナ**内に重ね、scrollTop の二重同期を使わない
- マスク部分クリック → そのマスクを削除
- デッキ選択（前回使用デッキをデフォルト）、任意メモ

## 編集追従

テキスト編集時、マスク range は `remapTextMasks` で再計算する。

- マスク内の追加・削除 → range が伸縮
- マスク外の編集 → 位置シフト
- マスク範囲をすべて削除 → マスク自体を除去

## キーボード

| キー | 動作 |
|------|------|
| ⌘Enter | 保存（または更新） |
| Enter | 改行（保存しない） |
| Esc | キャンセル（ウィンドウを閉じる） |

保存には **マスク 1 件以上** が必要。

## データ形式

```json
[{ "type": "range", "start": 12, "end": 18 }]
```

- `start` / `end`: UTF-16 コードユニットオフセット（JavaScript `selectionStart` 基準）
- 半開区間 `[start, end)`

## 再編集

ライブラリからカードクリック → `open_card_editor` → 同一エディタに `content` / `masks` / `note` をロード。

## 受け入れ条件

- [ ] マスク追加後、マスク内テキストを編集しても range が自然に追従する
- [ ] Enter で改行でき、⌘Enter のみで保存される
- [ ] マスククリックで解除、⌘Enter で保存後ライブラリに反映される
