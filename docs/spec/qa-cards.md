# 一問一答（Q&A）カード

## 概要

テキスト取込エディタで **全文** を問題文とし、別欄に **解答** を入力して保存するカード種別（`kind=qa`）。

- 問題文への **マスクは任意**（0 件でも保存可）
- 保存条件: **問題文 + 解答** 必須
- 既存 `text` / `image` カードは変更なし

## エディタフロー

1. ⌥⌘M → テキスト取込 → テキストマスクエディタ
2. ツールバー（メモ横）の **「Q」** → 現在の全文が問題文になる
3. レイアウトが Q&A モードへ切替。**上**: 問題文（マスク可能） / **下**: 解答 textarea
4. ⌘Enter で保存 → `kind=qa`

選択ポップアップには **「+ マスク」のみ**（Q は出ない）。

通常モード（Q 未使用）では従来どおり `kind=text`（マスク 1 件以上必須）。

## データ

| 列 | 用途 |
|----|------|
| `content` | 問題文 |
| `answer` | 解答 |
| `masks` | 問題文への range マスク JSON（`[]` 可） |

## 学習表示

### フラッシュカード / 学習(SRS)

| 状態 | 表示 |
|------|------|
| 非表示 | 問題文。マスクありなら Chartreuse 塗り |
| 答え表示 | 問題文は **全マスク解除** + 解答ブロックを下に表示 |

### 書く

- prompt: 問題文（マスク箇所は `【  】`）。マスク無しは問題文全文
- 正解: `card.answer`

### テスト

- 問題文（マスク反映）を stem、正解 = `card.answer`

### マッチ

- 問題文 ↔ `card.answer` のペア

## ライブラリ

- 種別バッジ: **Q&A**
- プレビュー: 問題文（マスク適用）。解答は非表示

## キーボード

| キー | 動作 |
|------|------|
| ⌘Enter | 保存 |
| Esc | キャンセル |

## 受け入れ条件

- [ ] 選択 → Q → 解答入力 → 保存（マスク無し可）
- [ ] 問題文マスク → 保存 → ライブラリ/再編集で一致
- [ ] 学習: Space → マスク解除 + 解答表示
- [ ] 書く/テスト/マッチに Q&A が含まれる
