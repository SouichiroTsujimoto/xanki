# 一問一答（Q&A）カード

## 概要

テキスト取込エディタで **全文** を問題文とし、別欄に **解答** を入力して保存するカード種別（`kind=qa`）。

- 問題文への **マスクは任意**（0 件でも保存可）
- 保存条件: **問題文 + 解答** 必須
- 既存 `text` / `image` カードは変更なし

## エディタフロー

1. ⌥⌘M → テキスト取込 → テキストマスクエディタ（またはデッキ学習のインライン作成）
2. ツールバーの **マスク | 一問一答** セグメントトグルでモードを切替
3. **一問一答** 選択時: レイアウトが Q&A モードへ。**上**: 問題文（マスク可能） / **下**: 解答 textarea
4. ⌘Enter で保存 → `kind=qa`（マスクモードでは `kind=text`）

選択ポップアップには **「+ マスク」のみ**（モード切替はツールバーのみ）。

マスクモードでは従来どおり `kind=text`（マスク 1 件以上必須）。

### モードの保持

- ユーザーがトグルで選んだモードはクライアント `localStorage`（`xanki:text-editor-mode`: `mask` | `qa`）に保存する
- **新規作成**（ウィンドウ / インライン）は前回選んだモードで開く（未設定時はマスク）
- **既存カードの再編集**はカードの `kind` を優先する（preference は上書きしない）
- 保存後の連続作成（`resetDraft`）も preference を復元する
- AI 候補の適用で一時的に Q&A になっても preference は変えない

## データ

| 列 | 用途 |
|----|------|
| `content` | 問題文 |
| `answer` | 解答 |
| `masks` | 問題文への range マスク JSON（`[]` 可） |

## 学習表示

### フラッシュカード / 学習(SRS)

| 状態 | 表示 |
|------|------|
| 非表示 | 問題文。マスクありなら Chartreuse 塗り |
| 答え表示 | 問題文は **全マスク解除** + 解答ブロックを下に表示 |

### 書く

- prompt: 問題文（マスク箇所は `【  】`）。マスク無しは問題文全文
- 正解: `card.answer`

### テスト

- 問題文（マスク反映）を stem、正解 = `card.answer`

### マッチ

- 問題文 ↔ `card.answer` のペア

## ライブラリ

- 種別バッジ: **Q&A**
- プレビュー: 問題文（マスク適用）。解答は非表示

## キーボード

| キー | 動作 |
|------|------|
| ⌘Enter | 保存 |
| Esc | キャンセル |

## 受け入れ条件

- [ ] トグルで一問一答 → 解答入力 → 保存（マスク無し可）
- [ ] 問題文マスク → 保存 → ライブラリ/再編集で一致
- [ ] 学習: Space → マスク解除 + 解答表示
- [ ] 書く/テスト/マッチに Q&A が含まれる
- [ ] 一問一答を選んで閉じたあと、次回の新規作成でも一問一答で開く
- [ ] 既存 text / Q&A カードの再編集は kind に従い、preference に引きずられない
