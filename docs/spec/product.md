# プロダクト

- **名称**: xanki（仮称、正式名称未定）
- **対象**: macOS ネイティブ（Apple Silicon / macOS 14+ 想定）
- **版**: v0.1（MVP 実装済み・安定化フェーズ）

## コンセプト

**「赤シート暗記」のデジタル化。** 画面に映っている Web・PDF・スライド等を、その場でマスキングして暗記カードにする。

既存 AI 暗記アプリの「素材丸投げで自動生成（代行）」に対し、**ユーザーが覚えたい箇所を自分で隠す**学習行為の摩擦を除去する。

## 設計原則

1. **速度がプロダクト** — ショートカットからマスクエディタ表示までの体感速度が最重要
2. **クラウド SSoT** — カード・デッキ・復習状態は D1 が正本。**Web / Desktop ともログイン必須**。Desktop は画像キャッシュのみローカル（[cloud.md](./cloud.md)）
3. **教材のいる場所にアプリが来る** — メニューバー常駐、ショートカット呼び出し
4. **AI・課金はクラウド層** — サーバー経由。詳細は [cloud.md](./cloud.md)

### 当初設計からの主な拡張

- 学習: デッキ学習（4 手段）+ スマート学習（SRS）（[study.md](./study.md)）
- デッキ JSON エクスポート/インポート
- ライブラリからカード編集（マスクエディタ再オープン）

## 明示的な非目標（MVP 外）

AI 機能(ローカル)、未ログイン利用、オフライン CRUD、共有、Windows/Linux、モバイル、プッシュ通知、自動アップデート。

## 受け入れ条件

- [ ] Google ログイン後、⌥⌘M / ⌥⌘S から各エディタが開き、カードを 1 枚以上保存できる
- [ ] 保存したカードがホーム / デッキ学習に表示され、学習モードで復習できる
- [ ] Web と Desktop で同一アカウントのデータが REST + SSE で反映される
