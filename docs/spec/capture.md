# 取込 (Capture)

## グローバルショートカット

| ショートカット | 機能 |
|--------------|------|
| ⌥⌘M | 選択テキスト取込 → テキストマスクエディタ |
| ⌥⌘S | スクショ取込 → 画像マスクエディタ |

Tray 常駐中に `tauri-plugin-global-shortcut` で登録。

## テキスト取込 (F2)

### フロー

1. ユーザーが任意アプリで文字列を選択し ⌥⌘M
2. Rust: クリップボード退避 → ⌘C 送出（`enigo`）→ 待機 → 読取 → 復元
3. 取得成功でテキストマスクエディタを新規ウィンドウで開く
4. 失敗時: 通知で「スクショ取込 (⌥⌘S)」へ誘導

### 権限

- **アクセシビリティ** — ⌘C 送出に必須

### 制約

- クリップボード退避・復元は必須（ユーザーのコピー内容を破壊しない）
- パスワードマネージャ等では失敗しうる → フォールバック導線を常に用意

## スクショ取込 (F3)

### フロー

1. ⌥⌘S
2. `screencapture -i -x <path>` で OS 標準の範囲選択 UI
3. 成功時: `App Data/images/{uuid}.png` に保存
4. 画像マスクエディタを新規ウィンドウで開く（**OS が選んだ範囲の PNG** を表示。エディタ内の範囲指定は行わない）

### エディタの役割

- 範囲選択は OS が完了済み
- エディタでは **矩形マスク配置のみ**（OCR UI は当面非表示。§OCR）
- 1 枚のスクショから 1 枚のカードを作成

### 権限

- **画面収録** — 必須。未許可時はオンボーディング/設定へ誘導

### 抽象化

キャプチャ処理は `CaptureProvider` として分離。将来自前 UI に差し替え可能。

## 受け入れ条件

- [ ] Safari/Chrome でテキスト選択 → ⌥⌘M → エディタに本文が入る
- [ ] ⌥⌘S → OS 範囲選択 → 画像エディタで即マスク可能
- [ ] 取込後、クリップボードの元内容が復元されている（テキスト取込）
